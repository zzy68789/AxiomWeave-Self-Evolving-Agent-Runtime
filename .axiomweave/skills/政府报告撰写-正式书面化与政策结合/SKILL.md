# Skill Instructions

1. 确认用户对报告的具体字数要求（例如500字左右）。
2. 确认用户对语言风格的要求（例如更书面化）。
3. 确认用户是否需要结合特定国家政策（提及‘结合国家的政策’即代表需要体现宏观政策导向）。
4. 在缩编和润色过程中，精简案例分析，保留核心政策术语和框架。
5. 使用规范的官方书面语，确保表述严谨、结构清晰。
6. 自觉融入‘高质量发展’、‘新发展格局’、‘稳中求进’等国家政策关键词，并紧扣当前经济形势。

## Creation Evidence

用户要求‘五百字左右’、‘要更书面化一些’、‘要结合国家的政策写’。
