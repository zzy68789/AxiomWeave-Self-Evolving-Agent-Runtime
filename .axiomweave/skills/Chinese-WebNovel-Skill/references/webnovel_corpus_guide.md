# 网文语料检索与模仿指南

这份文件只做一件事：帮你快速找到可借鉴的本地范本，并把“检索 -> 归纳 -> 回查原文 -> 生成新文本”这条链路走顺。

不要拿它抄句子。要拿它拆结构、拆节奏、拆信息投放。

## 一、当前本地语料

- 文章摘要：`../analysis/article_profiles.csv`
- 摘录库：`../analysis/excerpts.csv`
- 总索引：`../analysis/imitation_index.md`
- 原文目录：`../data/articles/`
- 元数据：`../data/metadata.csv`

当前已整理：

- `166` 篇文章
- `664` 条摘录
- 固定四类摘录：
  `开头钩子 / 主角亮相 / 高张力对白 / 结尾余韵`

## 二、什么时候先查语料

只要任务涉及这些内容，默认先查：

- 开头怎么抓人
- 某种题材怎么起势
- 某种关系流怎么写对白
- 章末怎么拉下一章
- 要求“像盐选风”“像真假千金文”“像校园甜虐文”
- 要求“重写我的开头”“给我几个同类写法”

不要等用户明确说“请模仿”才查。

## 三、检索脚本怎么用

脚本：

- `../scripts/search_corpus_examples.py`

先看有哪些可用项：

```bash
python3 scripts/search_corpus_examples.py --list-tags
python3 scripts/search_corpus_examples.py --list-types
python3 scripts/search_corpus_examples.py --list-keyword-examples
```

常用搜法：

```bash
python3 scripts/search_corpus_examples.py --type '开头钩子' --tag '危机压身' --limit 5
python3 scripts/search_corpus_examples.py --type '开头钩子' --tag '身份反差' --limit 5
python3 scripts/search_corpus_examples.py --type '高张力对白' --tag '关系破裂' --limit 5
python3 scripts/search_corpus_examples.py --type '结尾余韵' --tag '危机感' --limit 5
python3 scripts/search_corpus_examples.py --keyword '真假千金' --limit 10
python3 scripts/search_corpus_examples.py --keyword '弹幕' --limit 10
```

## 四、当前可用检索项

### `--tag`

这是固定标签集合，适合先按题材、关系、结构缩小范围。

题材/主题标签：

- `甜虐关系`
- `其他`
- `都市婚恋`
- `古言/宫廷`
- `弹幕/评论`
- `校园`
- `身份反差`
- `重逢修罗场`
- `系统/攻略`
- `危机压身`
- `仙侠/修仙`
- `奇幻/异种`
- `真假千金`
- `现实反差`

结构标签：

- `人设亮相`
- `主角亮相`
- `开头钩子`
- `结尾余韵`
- `高张力对白`
- `情感拉扯`
- `轻喜反差`
- `信息外挂`
- `关系破裂`
- `危机感`

### `--type`

这是固定摘录类型：

- `开头钩子`
- `主角亮相`
- `高张力对白`
- `结尾余韵`

### `--keyword`

这是自由搜索，不是固定白名单，但下面这些词目前命中率高，适合优先试：

- 设定词：`弹幕` `系统` `攻略` `穿书` `重生`
- 题材词：`修仙` `末世` `校园` `豪门` `古言`
- 关系词：`联姻` `和离` `离婚` `前夫` `竹马` `重逢`
- 冲突词：`真假千金` `恶龙` `河神` `缅北`

## 五、默认检索顺序

### 想仿开头

优先查：

- `--type '开头钩子'`
- 再叠加 `危机压身 / 身份反差 / 系统/攻略 / 校园 / 古言/宫廷`
- 不确定时再补 `--keyword`

看这些点：

- 第几段给异常局面
- 主角是不是马上进场
- 卖点是不是跟冲突绑在一起
- 有没有一上来就让读者知道“接下来会出事”

### 想仿对白

优先查：

- `--type '高张力对白' --tag '关系破裂'`
- `--type '高张力对白' --tag '情感拉扯'`

看这些点：

- 对白里有没有身份差
- 对白是不是句句有后果
- 这句对白有没有顺带改写关系状态

### 想仿章末

优先查：

- `--type '结尾余韵' --tag '危机感'`
- `--type '结尾余韵' --tag '情感拉扯'`

看这些点：

- 它停在决定、危险、关系变化还是半揭露真相上
- 它有没有把解释说满
- 读者为什么会自然点下一章

## 六、模仿流程

1. 先判断你要模仿的是什么，不要说“像某作者”，要说“像某种结构”。
2. 用 `--tag`、`--type`、`--keyword` 找 `2-4` 个近似例子。
3. 先读摘录，再回查原文上下文。
4. 总结共性。
5. 再生成新文本。

默认只总结这些可迁移信息：

- 第几段立冲突
- 主角怎么出场
- 信息怎么塞进动作或对白
- 这章停在什么变化上
- 同类套路最容易写假的地方是什么

不要总结：

- 作者爱用什么漂亮词
- 哪句话单独看最有感觉

## 七、怎么回查原文

每条摘录都能回到原文：

- 先在 `../analysis/imitation_index.md` 找 `excerpt_id`
- 再看对应标题和原文路径
- 最后去 `../data/articles/` 看上下文

需要上下文时，不要只盯摘录一句话。至少回看前后几段，判断：

- 这句是怎么被铺出来的
- 前面埋了什么信息
- 后面接了什么兑现

## 八、范例速查

### 危机压身型开头

- `A005`《酒午龙蛋》：联姻 + 恶龙契约 + 高代价关系
- `A083`《我在缅北做诈骗》：极端环境 + 主角反常态
- `A086`《急雨潮潮》：假死归来 + 旧账翻面

适合借鉴：

- 开头先扔代价，不先讲背景
- 危机和卖点绑在一起
- 主角一上来就已经处在压力里

### 身份反差型开头

- `A003`《贵族老登》：58 岁老登穿成万人嫌真千金
- `A031`《似曾相识燕归来》：重生后拒绝旧婚约
- `A047`《意外穿成了修仙文女配的剑灵。》：主角不是人，是剑灵

适合借鉴：

- 反差不要停留在设定介绍
- 反差要立刻转成局面冲突
- 开头最好让主角做一个和常规预期相反的动作

### 弹幕/外挂增压型开头

- `A001`《山雾零星》：脱离世界后被二次召回
- `A011`《春水泄愤》：弹幕把炮灰危机直接挑明
- `A055`《假千金竹马团轮流勾引我怎么办》：勾引、报复、反制
- `A066`《末世哭包最好命》：外挂及时改写行动路线
- `A072`《蛇仙许我三个心愿》：超自然承诺直接抬高后续期待

适合借鉴：

- 外挂信息必须改变主角当下判断
- 弹幕不是吐槽装饰，是剧情推进器
- 最好让外挂一出现就影响选择

### 关系破裂型对白

- `A002-dialogue`：古言关系里的规训压制
- `A012-dialogue`：看似稚嫩，实则强情感绑架
- `A056-dialogue`：句句捅现实命门

适合借鉴：

- 高张力对白不是骂狠，而是后果狠
- 说话方式要带身份感和利益感
- 一句对白最好同时推进冲突和关系变化

### 章末余韵型收尾

- `A016-ending`：强占有感的情绪余韵
- `A010-ending`：校园轻喜里的暧昧尾巴
- `A002-ending`：古言关系确认后的情绪回响

适合借鉴：

- 章末不一定靠大反转
- 关系状态一变，也能形成拉力
- 不要把该留到下一章的解释在本章说完

## 九、使用边界

- 允许模仿结构、节奏、信息投放、章末截断。
- 不要照抄原句，不要复制某个作者的表面文风。
- 不要把多个范本硬拼成“熟悉感大杂烩”。
- 检索的目的是学会“怎么写”，不是拼贴“像不像”。
