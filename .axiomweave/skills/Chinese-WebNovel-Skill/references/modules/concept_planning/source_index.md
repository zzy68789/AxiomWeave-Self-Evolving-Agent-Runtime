# 前置规划模块资料索引

## 一、本轮重组的主参考

- 主 `SKILL.md` 原有的“选材”和“构思”两大部分
- 主 `SKILL.md` 里原本关于 hook、premise、故事引擎、长度判断的全局规则

## 二、这轮模块化的目标

- 把主 skill 中过厚的前置规划内容下沉为独立模块
- 让主 `SKILL.md` 更接近“全局原则 + 模块路由”
- 让选材、构思、hook、premise、长度判断有单独的教程、运行规则、正例和反例

## 三、当前没有单独 raw 教程的说明

本模块这一轮主要来自 skill 内部方法重组，而不是用户单独收集的外部教程原件。

如果后续你补充了新的选材、hook、故事引擎教程，可以继续按其他模块的方式往这里补 `sources/raw/`。

## 四、与其他模块的衔接关系

- `concept_planning`：先判断点子值不值得写、骨架能不能跑
- `opening`：在骨架成立后处理开头成交
- `volume_outline`：在骨架成立后处理黄金三章、分卷和章纲
- `plot_logic / character_consistency / transition / dialogue / chapter_ending`：进入正文后处理局部执行问题
- `consistency_review`：章节完稿后统一复查
