# 前置规划反例库

## 快速索引

- A 组：题材和消费点混乱
- B 组：卖点和 hook 失焦
- C 组：主角处境空转
- D 组：故事引擎断电
- E 组：长度和卷一方向失衡

## A 组：题材和消费点混乱

### S-B01 什么都想要，结果什么都不清

```text
这是一本融合了都市、古言、悬疑、仙侠、校园和职场元素的成长小说。
```

问题：题材和消费点都被搅浑了，读者不知道该用什么预期进入。

### S-B02 只会说“很特别”

```text
女主是一个很特别的人，会很多技能，也经历了很多事。
```

问题：没有题材定位，也没有主卖点。

### S-B03 只有世界，没有吃点

```text
这个修仙世界里有很多门派、法器和历史传说。
```

问题：世界观可以存在，但读者还不知道这本书最先卖什么。

## B 组：卖点和 hook 失焦

### S-B04 最强卖点被埋没

```text
数百年前，王朝更迭，宗门四起，而她的命运也在这一天缓缓转动。
```

问题：信息很多，但没有把最抓人的异常局面推到前面。

### S-B05 只说会变，不说怎么变

```text
从那一天起，她的人生彻底不同了。
```

问题：空话句，没有异常局面，也没有动作。

### S-B06 一句话里堆太多方向

```text
她既要复仇、又要恋爱、还要拯救世界、破解阴谋、继承家业。
```

问题：卖点不是越多越好，主驱动力反而被冲淡了。

## C 组：主角处境空转

### S-B07 主角可以一直不动

```text
她虽然穿成了真千金，但暂时也没什么麻烦，先看看再说。
```

问题：没有眼前压力，故事不会自己启动。

### S-B08 第一章只有日常，没有任务

```text
男主醒来、吃饭、上班、下班，顺便介绍了自己的性格。
```

问题：读者还不知道第一波冲突是什么。

### S-B09 配角在动，主角没被咬住

```text
身边所有人都很紧张，但主角只是听他们讲了一通家族往事。
```

问题：局势很热闹，不等于主角处境成立。

## D 组：故事引擎断电

### S-B10 点子只有一次性反转

```text
她发现自己不是亲生的，真相揭开后，一切问题都解决了。
```

问题：这更像一个短梗，不像能持续推进的连载引擎。

### S-B11 只有大谜团，没有阶段推进

```text
整本书的目标就是查清二十年前那件事，中间暂时没有别的节点。
```

问题：没有阶段任务，读者很难撑过长线过程。

### S-B12 主角太顺，后面没阻力

```text
她一开局就拥有顶级资源、绝对实力和所有人的支持。
```

问题：没有持续阻力，后续章节会越来越空。

## E 组：长度和卷一方向失衡

### S-B13 一个短点子硬拉百万字

```text
她只想离一次婚，这个点子准备写成超长连载。
```

问题：单一目标太薄，支撑不起超长体量。

### S-B14 想写长篇，却没有卷一任务

```text
这本书后面会很精彩，但卷一先随便写着看。
```

问题：没有第一阶段目标，长篇只会越写越散。

### S-B15 体量建议只看作者意愿

```text
我很喜欢这个设定，所以它应该写成长篇。
```

问题：长度判断要看引擎和阶段变化，不看作者舍不舍得放下。
