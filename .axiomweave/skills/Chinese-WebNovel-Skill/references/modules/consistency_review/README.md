# 审查模块

这是一个专门用于章节完稿后做一致性审查的专项模块。

它不是用来写正文的，而是用来做最后一道质量闸门。

主要解决：

- 写完一章后没系统复查
- 转场顺了，但逻辑断了
- 逻辑通了，但人物崩了
- 情节在推，信息和状态在漏

## 什么时候调用

每一章写完之后，都默认调用本模块做审查。

不是“有问题再看”，而是“写完必看”。

## 建议阅读顺序

1. 先读 [tutorial.md](tutorial.md)。
2. 再读 [runtime.md](runtime.md)。
3. 按清单逐项自查。
4. 有问题时回跳到对应专项模块。
