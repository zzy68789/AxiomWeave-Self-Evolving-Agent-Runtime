# 人物一致性教程

## 一、先统一一个定义

人物一致性不是“人物永远不变”，而是：

`人物的变化要有来路，而且前后的目标、情绪、关系、身体和说话方式要能对得上。`

人物可以变，但不能无故跳。

## 二、人物一致性的五个核心维度

### 1. 目标一致

这一章里他最想要什么，不能一会儿一个方向。

### 2. 情绪一致

情绪可以变化，但要有触发物和过渡。

### 3. 关系一致

上一场刚撕破脸，下一场自然亲近，就需要中间机制。

### 4. 身体一致

伤、病、醉酒、疲惫、怀孕、失眠，这些都不能切场就忘。

### 5. 声音一致

同一个人说话习惯、出手方式、忍耐阈值，要相对稳定。

## 三、人物“变”也要讲逻辑

人物不是不能成长，不是不能黑化，不是不能心软。

但每一次大变化至少要有：

- 触发
- 过程
- 代价

## 四、最常见的人物一致性问题

### 1. 情绪翻面没来路

上一秒还恨，下一秒就原谅。

### 2. 关系状态被格式化

上一场刚闹翻，下一场又像没事人。

### 3. 身体状态丢失

刚受伤，下一场就活蹦乱跳。

### 4. 说话声音飘

前面冷硬寡言，后面突然变成长篇鸡汤输出机。

### 5. 目标断线

本来这一章要做 A，写到中间忽然开始做 B，而且没有转移原因。

## 五、写人物前的最小状态卡

动笔前至少写清：

- 他现在最想达成什么：
- 他现在最强烈的情绪是什么：
- 他和场上关键人物的关系温度是什么：
- 他身体上有什么不能忘：
- 他说话和行动最像什么风格：

如果这五句不清楚，人物很容易飘。
