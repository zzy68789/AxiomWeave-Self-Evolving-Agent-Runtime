# 人物一致性反例库

## 快速索引

- A 组：目标乱跳
- B 组：情绪跳变
- C 组：关系断片
- D 组：身体失忆
- E 组：声音飘移

## A 组：目标乱跳

### C-B01 主线突然蒸发

```text
上一场她还拼命想逃命。
下一场她忽然开始认真策划开店。
```

问题：目标切换没有触发，也没有承压过程。

### C-B02 为无关小事放掉主目标

```text
她前面还发誓要查真相。
后面因为一个配角难过，就把整条主线放下了。
```

问题：人物优先级像被作者临时改写。

### C-B03 刚立决心，下一场就忘

```text
他刚说无论如何都要离开这个家。
下一场又像无事发生一样坐下来吃饭。
```

问题：重要目标不可能隔一场自动失效。

## B 组：情绪跳变

### C-B04 突然原谅

```text
上一秒她还恨得发抖。
下一秒她忽然平静下来，决定原谅他。
```

问题：没有触发，没有过渡。

### C-B05 刚社死就开始打趣

```text
她前一场刚在所有人面前出丑。
下一场却已经能轻松开玩笑了。
```

问题：情绪恢复得像重启，不像人。

### C-B06 背叛冲击没有余波

```text
她刚得知闺蜜背叛自己。
转头就自然地替对方找借口。
```

问题：重大情绪冲击没有留下任何停滞和抗拒。

## C 组：关系断片

### C-B07 冷战一键恢复

```text
上一场他们刚撕破脸。
下一场他自然地替她拉开椅子，两人有说有笑。
```

问题：关系温度像被重置。

### C-B08 分手后亲昵称呼照旧

```text
前面已经正式分手。
后一场她还自然地叫他“宝宝”。
```

问题：称呼和关系状态不匹配。

### C-B09 刚站错队就并肩作战

```text
他上一场才公开偏向别人。
下一场她却毫无芥蒂地跟他联手。
```

问题：缺少重新建立信任的过程。

## D 组：身体失忆

### C-B10 伤口凭空消失

```text
上一场她刚被玻璃划破手掌。
下一场她稳稳当当地切菜。
```

问题：身体连续性丢失。

### C-B11 熬夜后精力满格

```text
他通宵逃亡一整夜。
第二天依然神清气爽地分析局势。
```

问题：疲劳没有留下任何成本。

### C-B12 高烧中长篇输出

```text
她烧到发抖。
却还能完整、有条理、毫不喘气地说一大段话。
```

问题：身体状态和台词表现互相打架。

## E 组：声音飘移

### C-B13 冷硬人忽然上心理课

```text
前面一直冷硬寡言的人，后面忽然开始长篇大论解释自己的童年创伤。
```

问题：不是不能说，而是缺少逼到他开口的条件和说话方式过渡。

### C-B14 粗人忽然文绉绉

```text
一直张口就是脏话的人，忽然说起了“人生的旷野与困境”。
```

问题：语言芯片突然被换了。

### C-B15 谨慎人忽然高调挑衅

```text
前面一向小心避事的人，后面没有任何铺垫就当众撕桌挑衅。
```

问题：说话方式和生存习惯一起崩了。
