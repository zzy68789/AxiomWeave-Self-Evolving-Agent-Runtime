# 网文开头教程

## 一、先统一一个定义

网文开头不是铺垫区，而是成交区。

它至少要尽快让读者知道三件事：

- 主角是谁
- 现在出了什么异常
- 这本书最值得追的东西是什么

好开头不是把设定讲清，而是把读者拉进局里。

## 二、开头最常见的四种成交方式

### 1. 危机压身

做法：

- 一上来就让主角处在要付代价的局里
- 卖点和危机绑定，不分家

适合：

- 仙侠
- 古言婚恋
- 悬疑求生
- 高压情感关系

### 2. 身份反差

做法：

- 让主角身份本身就是冲突
- 不先解释背景，先让反差落成局面

适合：

- 真假千金
- 穿书
- 重生
- 反套路轻喜

### 3. 信息外挂

做法：

- 用弹幕、系统、帖子、爆料、录音、诊断结果等外来信息改写主角对现实的理解
- 外挂不是装饰，而是改轨器

适合：

- 穿书
- 校园
- 都市关系流
- 真假千金

### 4. 关系炸点

做法：

- 不从天气和起床开始，而是直接从关系里最危险的刀口切入
- 让一句话、一个发现、一个举动马上改写关系位置

适合：

- 婚恋
- 家族撕裂
- 校园拉扯
- 重逢修罗场

## 三、判断开头写得好不好，不看文艺感，看这五件事

### 1. 前 300-500 字有没有抓手

如果前几百字还在正常生活、正常回忆、正常说明，通常就是慢。

### 2. 前 1500-3000 字有没有卖点

抓手不是卖点。

只有热闹，没有“这本书凭什么继续看”，也不行。

### 3. 主角是不是站在局里

好开头里的主角不是站在一边介绍故事，而是已经被卷进故事。

### 4. 信息是不是服务冲突

设定、背景、关系说明，都要服务当前冲突。

不能把开头写成世界观说明书。

### 5. 黄金三章有没有递进

第一章立异常。
第二章扩卖点。
第三章立方向和追更预期。

如果三章只是把第一章换个说法重复三遍，读者还是会走。

## 四、开头最常见的五种硬伤

### 1. 起得太正常

比如先起床、洗漱、吃饭、看天气。

### 2. 先讲设定，再讲故事

读者还没关心主角，就先被塞了一屏背景说明。

### 3. 只有钩子，没有卖点

看似很炸，但炸完不知道后面要看什么。

### 4. 只有卖点词，没有局面

比如说“她是真千金”“她重生了”“她觉醒了系统”，但没有立刻形成局面变化。

### 5. 主角被动站桩

什么事都发生了，但主角只是旁观和解释，没有动作，没有选择。

## 五、网文开头的最小计划

动笔前先写清这五句：

- 这本书开头第一刀切在哪：
- 主角当前最具体的麻烦是什么：
- 最早抛出的卖点是什么：
- 第一章结束前局面会怎么变：
- 第三章前要让读者形成什么追更问题：

如果这五句写不清，开头大概率会散。
