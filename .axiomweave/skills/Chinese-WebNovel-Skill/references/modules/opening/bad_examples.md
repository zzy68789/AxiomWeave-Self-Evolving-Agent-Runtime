# 开头反例库

使用方法：

1. 先看自己最像哪一类。
2. 再判断缺的是抓手、卖点还是局面。
3. 回到 `runtime.md` 决定怎么补。

## 快速索引

- A 组：慢热铺垫型失败
- B 组：有事件但没有卖点
- C 组：设定说明书型失败
- D 组：热闹很大但主角不进局

## A 组：慢热铺垫型失败

### O-B01 从天气和起床开始

```text
那天天气很好。
她像往常一样起床、洗漱、吃早饭，然后去上学。
```

问题：没有异常，没有抓手，没有理由继续读。

### O-B02 先回忆三段过去

```text
她从小就是个很善良的人。
小时候，母亲经常告诉她……
```

问题：作者在介绍人物，不是人物在进局。

### O-B03 先写环境，再写环境

```text
院子里花开得很好，风很轻，天很蓝，窗帘也被吹得轻轻晃动。
```

问题：文字在动，故事没动。

### O-B04 先写日常流程

```text
她下楼买豆浆，路过早餐摊，又顺便去拿快递。
```

问题：这不是不能写日常，而是日常里没有异常事件。

## B 组：有事件但没有卖点

### O-B05 开头很吵，但不知道这本书卖什么

```text
宴会厅里忽然停电。
所有人都尖叫起来。
她也吓了一跳。
```

问题：只有热闹，没有题材引擎。

### O-B06 只抛一个秘密，不给后续方向

```text
她发现丈夫出轨了。
她的心一下子碎掉了。
```

问题：冲突有了，但没看见这本书后面怎么跑。

### O-B07 只有事故，没有主角选择

```text
飞机忽然剧烈颠簸，氧气面罩掉了下来。
她坐在原地，不知所措。
```

问题：场面很大，但和这本书的卖点、主角能力、后续路线都没接上。

### O-B08 误会起了，但没引擎

```text
所有人都误会她偷了项链。
她哭着跑了出去。
```

问题：如果只剩“被误会”，却没有反打或后续发动机，开头很快会塌。

## C 组：设定说明书型失败

### O-B09 先讲世界规则三百字

```text
这个大陆分为东境、西境、南境和北境。
每个境又分九阶灵脉……
```

问题：读者还没认识主角，先被规则埋了。

### O-B10 先说自己是重生者，但局面没变

```text
我重生了。
上辈子我很惨。
这辈子我要改变命运。
```

问题：只有设定标签，没有当前场面。

### O-B11 外挂先自我介绍

```text
系统是一种高维智能体，主要负责……
```

问题：外挂还没改剧情，就先开始讲产品说明。

### O-B12 先堆名词不给处境

```text
我是SSS级向导，联邦军校首席，精神海评级异常……
```

问题：名词很多，但主角眼前到底有什么麻烦看不见。

## D 组：热闹很大但主角不进局

### O-B13 所有人都在动，主角只会看

```text
四周乱成一团。
她站在那里，心想事情为什么会变成这样。
```

问题：主角不在局里，只在旁边感慨。

### O-B14 外挂出现了，但没改选择

```text
眼前出现了弹幕。
她看了一会儿，决定还是先去上课。
```

问题：外挂没有改轨，只像装饰品。

### O-B15 围攻很大，主角零反应

```text
众人都在指责她。
她只是呆呆站着，一句话也不说。
```

问题：如果主角连最基本的受压反应都没有，开头会显得发木。

### O-B16 关系炸点起了，但没有立场

```text
前男友突然回来了。
她心里很乱。
```

问题：关系事件已经够大，但主角没有立即形成态度、目标或代价。
