# 开头模块来源索引

## 一、主框架来源

本模块的主判断框架，主要来自：

- 主 `SKILL.md` 里“第三部分：开头”
- 主 `SKILL.md` 里“第四部分：黄金三章”
- 主 `SKILL.md` 里“第一部分：选材”与“第二部分：构思”中关于卖点、冲突、故事引擎的要求

## 二、主要语料摘录来源

本模块正例主要取自 `analysis/excerpts.csv` 中这些开头钩子摘录：

- `A001-opening`《山雾零星》：信息外挂改轨
- `A003-opening`《贵族老登》：身份反差
- `A005-opening`《酒午龙蛋》：危机压身
- `A010-opening`《抓走了校霸的戒指怎么办》：帖子式轻喜抓手
- `A014-opening`《天才小师妹考试失利。》：群体围攻反打
- `A016-opening`《吻颜》：高代价关系局

## 三、这轮模块重组的原则

- 不再把“开头”只看成某几种题材套路。
- 优先按“抓手 / 卖点 / 局面 / 黄金三章递进”来组织。
- 正例优先选那种一上来就能成交的摘录。
- 反例优先写最容易让读者滑走的失败类型。
