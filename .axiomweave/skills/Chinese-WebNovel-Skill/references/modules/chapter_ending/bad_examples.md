# 章末反例库

## 快速索引

- A 组：总结和讲理型
- B 组：情绪散尽型
- C 组：假悬念型
- D 组：上一章和下一章断线型

## A 组：总结和讲理型

### E-B01 人生感悟型

```text
她想，人生果然充满了无奈。
```

问题：只有感想，没有后劲。

### E-B02 主题升华型

```text
所以我们都要学会珍惜眼前人。
```

问题：像作文结尾，不像网文章末。

### E-B03 作者替读者总结型

```text
这一晚之后，他们之间的关系变得更加复杂了。
```

问题：只是告诉读者“复杂了”，没有把复杂落在动作或决定上。

### E-B04 讲道理收束型

```text
她终于明白，感情里最重要的还是沟通。
```

问题：人物刚经历大事，章末却收成了一句普世道理。

## B 组：情绪散尽型

### E-B05 冲突后拖日常

```text
他们大吵了一架。
后来她洗了澡，吹干头发，喝了杯水，慢慢睡着了。
```

问题：最值钱的情绪已经过了，尾巴还在拖。

### E-B06 哭完收干净

```text
她哭了很久，最后终于释怀，安心地闭上眼睛睡了。
```

问题：情绪被一次性放完了，没有给下一章留余震。

### E-B07 解释过量型

```text
他走后，她坐在床边想了整整一页，分析自己为什么会这么难过。
```

问题：把情绪全部讲清楚了，章末的模糊拉力反而没了。

### E-B08 把力耗在流程上

```text
她收拾好桌子，关了灯，给手机充上电，钻进被窝。
```

问题：如果前面刚有冲击，章末却掉进生活流程，力会迅速泄掉。

## C 组：假悬念型

### E-B09 自己拆自己的钩

```text
门外传来一阵敲门声。
她吓了一跳。
原来是外卖到了。
```

问题：刚起的力被自己拆掉了。

### E-B10 装神秘但不重要

```text
手机忽然亮起。
她心头一紧。
原来是运营商欠费提醒。
```

问题：形式像钩子，内容却完全不改局面。

### E-B11 硬停在半句

```text
她打开信封，只看到第一行字，就猛地睁大了眼睛。
```

问题：如果前面没有足够铺垫，这种“瞪眼停住”只会显得机械。

### E-B12 纯为吊胃口而吊

```text
而这一切，不过只是开始……
```

问题：没有具体变化支撑，像通用尾巴。

## D 组：上一章和下一章断线型

### E-B13 决定句无人接

```text
他终于决定去见她。

下一章：
第二天阳光很好，他先去买了早餐。
```

问题：上一章的决定没有被下一章第一拍接住。

### E-B14 危机句隔章蒸发

```text
她看见了那份离婚协议。

下一章：
公司开会，她先汇报了季度数据。
```

问题：本该最先处理的冲击被换成了别的事。

### E-B15 关系改写句无后果

```text
「从今天起，我们两清。」

下一章：
两人见面像没事人一样吃饭。
```

问题：章末明明改了关系，下一章却像没发生。

### E-B16 章末埋雷，下章绕开

```text
短信只有四个字：我怀孕了。

下一章：
她先去了趟超市买水果。
```

问题：如果下一章第一拍不先处理这颗雷，上一章章末等于白收。
