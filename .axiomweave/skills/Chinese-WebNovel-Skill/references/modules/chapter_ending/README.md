# 章末模块

这是一个专门处理网文章末问题的专项模块。

它主要解决：

- 章末收得太平
- 章末停在总结和讲道理
- 章末刚蓄力，下一章接不住
- 章末有悬念，但像硬钩
- 章末没有后劲

## 什么时候调用

当任务明确涉及这些问题时，优先调用本模块：

- 用户要你写章末、补钩子、改结尾
- 用户说“这一章收不住”“没追更欲”
- 章末停在解释、抒情、总结
- 上一章和下一章之间的拉力断掉

## 建议阅读顺序

1. 先读 [tutorial.md](tutorial.md)。
2. 再读 [runtime.md](runtime.md)。
3. 去 [good_examples.md](good_examples.md) 选 `2-4` 个正例。
4. 去 [bad_examples.md](bad_examples.md) 选 `1-2` 个反例。
5. 先定章末落点，再写正文。
