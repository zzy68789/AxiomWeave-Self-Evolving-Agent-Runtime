# 章末模块来源索引

## 一、主框架来源

本模块主要来自：

- 主 `SKILL.md` 里“第八部分：章末”
- 主 `SKILL.md` 里“逐章写作”与“转场模块”中关于章节余力和下一场承接的要求

## 二、主要语料摘录来源

本模块正例主要取自 `analysis/excerpts.csv` 中这些结尾余韵摘录：

- `A001-ending`：含笑落泪型余韵
- `A002-ending`：柔收型关系落点
- `A010-ending`：轻喜回钩
- `A016-ending`：长期张力承诺
- `A056-ending`：动作型焚毁旧物
- `A064-ending`：重逢反打
- `A086-ending`：一句决定带团圆

## 三、这轮重组原则

- 不把章末只看成“最后一句漂亮话”。
- 优先按“余韵 / 回钩 / 决定 / 承接”来组织。
- 强调章末必须和下一章天然相连。
