# 转场反例库

这份反例库也已经按新版模块结构重编。

现在不是 20 条平铺，而是按“错误簇”分组。

使用方法：

1. 先判断自己当前最像哪一类错误。
2. 再看这类错误通常断在什么地方。
3. 回到 `runtime.md`，决定该补后果、桥梁动作、共同事件还是边界提示。

## 快速索引

- A 组：完全没搭桥的新手硬切
- B 组：时间桥、空间桥、章末承接没搭稳
- C 组：人物状态、情绪、关系、身体连续性断片
- D 组：视角、回忆、梦境边界失控
- E 组：想写巧却没有借力点的高阶误用

## A 组：新手型硬切

这组最常见，也最容易修。

它们通常不是“写得不高级”，而是压根没搭桥。

### T-B01 只有时间词，没有后果

```text
第二天到了。
她起床去上班。
```

问题：只报时，不报变。

### T-B02 只说到了某地，没有第一眼

```text
到了医院。
她下车后走了进去。
```

问题：地点变了，但剧情没有新东西。

### T-B03 环境铺太长，不见事

```text
夜色温柔，月光如纱，走廊很长，墙壁泛着冷白的光。
她一步一步向前走去，心里想了很多。
```

问题：场景在动，剧情没动。

### T-B04 作者总结替代桥梁动作

```text
总之，这件事让他们之间的关系变得微妙起来。
```

问题：只有结论，没有现场。

### T-B05 滥用“与此同时”

```text
与此同时，她回到了宿舍。
与此同时，他还在楼下等她。
与此同时，弹幕也炸了。
```

问题：词在切，镜头没组织好。

## B 组：时间与空间桥没搭稳

这组问题的特点是：

- 看起来写了转场
- 但读者还是会问“然后呢”“怎么就到这了”

### T-B06 时间跨度很大，但没有阶段节点

```text
后来过了很多年，他们都长大了，生活也变了很多。
```

问题：全是总结，没有命运拐点。

### T-B07 接到电话，下一场直接到医院

```text
电话那头说孩子进医院了。
下一段她已经坐在病房里。
```

问题：人物反应和赶过去的逻辑全没了。

### T-B08 移步换景没有动词链

```text
山顶很高，大殿很亮，池塘很香，风景很好。
```

问题：空间信息并列堆放，没有“走过去”的感觉。

### T-B09 章末刚吊悬念，下章先写早餐

```text
门外传来敲门声。

第二天阳光很好，她吃了两个包子。
```

问题：上一章的力全跑了。

### T-B10 悬念刚立刻就解释没了

```text
她看见床下伸出一只手。
当然，那只是弟弟在找拖鞋。
```

问题：张力还没形成就自拆。

## C 组：人物和情绪断片

这组是网文里最伤的一类，因为它不只是转场硬，还会把人物写崩。

### T-B11 情绪突然翻面

```text
上一秒她还恨得发抖。
下一秒她忽然平静下来，决定原谅他。
```

问题：人物状态断裂。

### T-B12 上一场还抗拒，下一场直接配合

```text
她明确说自己不会去见他。
转头就出现在了他的生日宴上。
```

问题：缺了承压或逼迫机制。

### T-B13 场景切了，身体状态忘了

```text
上一场她刚被玻璃划破手掌。
下一场她端着热汤稳稳当地切菜。
```

问题：人物身体连续性丢了。

### T-B14 场景切了，关系状态忘了

```text
上一场他们刚撕破脸。
下一场他自然地替她拉开椅子，两人有说有笑。
```

问题：关系像被格式化。

### C 组这类该补什么

- 情绪触发物
- 逼迫机制
- 伤势或身体反应
- 关系缓和或恶化的中间一步

## D 组：视角、回忆与梦境边界失控

这组问题最容易让读者迷路。

### T-B15 回忆没有触发物

```text
她坐在沙发上。
忽然想起十年前的夏天。
```

问题：说进回忆就进，边界太虚。

### T-B16 从回忆回来没有边界

```text
那年他对她笑得很好看。
她低头喝了口水。
```

问题：读者不知道已经回到现实。

### T-B17 视角突然换人

```text
她在楼下等车。
与此同时，城北的男人正在抽烟。
```

问题：作者想切，不是剧情需要。

### T-B18 同段混多个脑内视角

```text
她觉得他在撒谎，而他也知道她已经起疑，司机则担心今天会出事。
```

问题：镜头乱，读者没有稳定站位。

### T-B19 梦境与现实边界糊掉

```text
他看见她满身是血地朝自己走来。
电话响了，他去开门，门外是她。
```

问题：前后是不是同一层现实不清楚。

## E 组：高阶方法误用

这组不是“完全不会写”，而是想写巧，结果写虚了。

### T-B20 物件突然重要，但从没埋过

```text
她握紧那支意义非凡的钢笔，决定离开。
```

问题：物件没有前史，借力点根本不存在。

### E 组这类该补什么

- 物件前史
- 共同触点
- 过渡中的可见联系
- 让“高级感”服从读者理解，而不是反过来
