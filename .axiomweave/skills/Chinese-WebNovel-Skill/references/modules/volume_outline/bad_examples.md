# 分卷章纲反例库

## 快速索引

- A 组：前三章失效
- B 组：第一卷失焦
- C 组：章纲流水账
- D 组：中段发散

## A 组：前三章失效

### V-B01 先讲背景不成交

```text
前三章还在解释世界规则和人物童年。
```

问题：没有成交，只有准备。

### V-B02 三章都在重复同一场情绪

```text
第一章她委屈。
第二章她还是委屈。
第三章她继续委屈。
```

问题：情绪在重复，局面没递进。

### V-B03 第三章还没给方向

```text
读完第三章，只知道主角很惨，不知道她接下来要干什么。
```

问题：黄金三章没有把追更问题立起来。

### V-B04 卖点到第三章还没露头

```text
简介里说有弹幕、有系统、有真假千金。
结果前三章都还没真正触发这些东西。
```

问题：题材消费点埋得过深，读者在前几章吃不到货。

## B 组：第一卷失焦

### V-B05 第一卷什么都想写

```text
第一卷什么都想写，最后没有核心兑现点。
```

问题：卷没有中心。

### V-B06 没有主压迫源

```text
这一卷今天打脸这个，明天认识那个，后天去新地图，但没有一个持续压着主角的问题。
```

问题：卷像散点合集，不像完整阶段。

### V-B07 卷末自然停，没有换局

```text
卷末只是“这一段先写到这里”，没有身份、关系或资源的大变化。
```

问题：读者感觉不到“这一卷值了”。

### V-B08 卷目标说得太虚

```text
第一卷目标：成长、变强、变优秀。
```

问题：太抽象，无法指导章节设计。

## C 组：章纲流水账

### V-B09 只记发生了什么

```text
第 1 章相遇。
第 2 章吃饭。
第 3 章聊天。
```

问题：记录了事件，没有记录更新价值。

### V-B10 每章都一个强度

```text
每章都是“得知一点消息，再多聊几句，再回家想想”。
```

问题：章纲没有节奏起伏。

### V-B11 没有单章目标

```text
这一章主要写他们关系推进。
```

问题：太虚，正文时很容易发空。

### V-B12 章末不给拉力

```text
每章都收在“今天先这样吧”。
```

问题：章纲层就没把追更欲设计进去。

## D 组：中段发散

### V-B13 刚到主线关键处就开三条支线

```text
刚写到主线关键处，就岔去开三条支线，谁也不推进。
```

问题：没有卷目标约束，故事会越来越松。

### V-B14 新人物只增加热闹

```text
中段不断加新角色，但谁也没带来新压力、新资源或新信息。
```

问题：表面变丰富，结构上却是空转。

### V-B15 小高潮一直不来

```text
主线拖了十几章，始终没有一个阶段性兑现点。
```

问题：读者会觉得一直在蓄力，却没回报。

### V-B16 舍不得删重复戏

```text
同一种试探、同一种嘴炮、同一种误会连续写了四五章。
```

问题：中段发散往往不是因为戏少，而是重复戏太多。
