# 分卷章纲模块

这是一个专门处理长线规划、分卷设计、黄金三章递进和章纲拆分的专项模块。

它主要解决：

- 开了头，但后面怎么跑不清
- 黄金三章有热闹，但没有后续引擎
- 分卷散
- 中段发散
- 章纲只有事件，没有拉力
- 卷末没有兑现点

## 什么时候调用

当任务明确在处理这些问题时，优先调用本模块：

- 用户要你做分卷
- 用户要你写前 10-20 章章纲
- 用户说“中段容易散”“不知道后面怎么跑”
- 已经有 premise，但还没有连载结构

## 建议阅读顺序

1. 先读 [tutorial.md](tutorial.md)。
2. 再读 [runtime.md](runtime.md)。
3. 去 [good_examples.md](good_examples.md) 看结构正例。
4. 去 [bad_examples.md](bad_examples.md) 看最常见的散法。
5. 需要落成可复用大纲时，再对照 [outline_template.md](outline_template.md) 和 [chapter_template.md](chapter_template.md)。
6. 先写卷级计划，再拆章纲。

## 文件说明

- [tutorial.md](tutorial.md)
  把黄金三章、分卷、章纲、卷末兑现统成一套长线结构方法。

- [source_index.md](source_index.md)
