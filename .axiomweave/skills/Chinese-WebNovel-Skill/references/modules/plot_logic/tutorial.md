# 剧情逻辑教程

## 一、先统一一个定义

剧情逻辑不是“现实里会不会发生”，而是：

`在这个故事已经给出的前提下，这件事为什么会发生，而且为什么偏偏在这里发生。`

读者说“逻辑不通”，很多时候不是说不可能，而是说：

- 你没给前提
- 你没给触发
- 你没给代价
- 你没给后果

## 二、剧情逻辑的五个核心环节

### 1. 动机

任何大事件都要先回答：

- 人物想得到什么
- 不做会失去什么
- 这件事为什么值得他付成本

### 2. 触发

事情不会凭空发生。

要有让人物“现在必须这么做”的那一下。

### 3. 决策

人物做出的选择要和他的处境、性格、目标匹配。

### 4. 后果

选择之后，局面必须变。

没有后果，剧情就像没发生。

### 5. 兑现

前面种下的东西，后面要以冲突、反转、代价、收益中的某一种方式兑现。

掉马、翻案、误会、打脸这类高刺激节点，本质上都属于兑现的一部分。

## 三、最常见的剧情逻辑问题

### 1. 动机空

人物要做一件大事，但读者不知道他为什么非做不可。

### 2. 触发弱

事情会发生，但读者不知道为什么偏偏是现在。

### 3. 决定跳

上一场还抗拒，下一场突然全力配合。

### 4. 事件硬降临

关键消息、关键人物、关键道具刚好出现得太巧。

### 5. 误会和反转没前提

只为了制造爽点或刀点，没做铺垫。

### 6. 发生了，但局面没变

不是没有事件，而是事件没改写关系、资源、风险和下一步压力。

## 四、剧情逻辑不通时，不要先润色

先查这五件事：

- 动机在哪里
- 触发在哪里
- 人物为什么这么选
- 选完以后局面变了什么
- 这件事有没有为后文留下可兑现的东西

## 五、写剧情前的最小因果链

动笔前先写清：

- 现在人物最想得到什么：
- 什么东西逼他现在动：
- 他准备怎么做：
- 做了以后最直接的后果是什么：
- 这个后果会把下一章推向哪里：

如果这五句写不清，逻辑很容易松。
