# 剧情逻辑反例库

## 快速索引

- A 组：动机不成立
- B 组：触发凭空降临
- C 组：决策不匹配
- D 组：反转和误会硬翻
- E 组：发生了但没变化

## A 组：动机不成立

### P-B01 无压硬送命

```text
她明知道去会死，也没有任何更大压力，还是决定独自赴约。
```

问题：没有足够欲望或逼迫机制。

### P-B02 只因“好奇”就冒巨大险

```text
她只是有点好奇，就决定夜闯对家地下室。
```

问题：风险和动机完全不成比例。

### P-B03 主角明明不缺这个，却偏要抢

```text
他既不缺钱，也不缺资源，却为了一个普通项目突然翻脸。
```

问题：看不出这件事为什么值得他付出这么大成本。

## B 组：触发凭空降临

### P-B04 工具人式真相投递

```text
她正发愁，门外忽然来了一个神秘老人，正好知道全部真相。
```

问题：工具人式降临，像作者伸手。

### P-B05 需要证据时证据刚好出现

```text
她正想证明清白，监控录像忽然自动恢复了。
```

问题：触发服务作者需求，不服务故事世界本身。

### P-B06 倒计时从天而降

```text
一直没人提过截止时间。
到这章忽然说明天就是最后期限。
```

问题：紧迫感是临时加的，不是前面长出来的。

## C 组：决策不匹配

### P-B07 性格保守的人突然莽到底

```text
前面一直谨慎的人，后面没有任何铺垫就决定直播掀桌。
```

问题：选择方式不像这个人。

### P-B08 明明有更稳的路不走

```text
她手里有证据、有律师、有证人，却还是选择独自闯过去对骂。
```

问题：文本没解释为什么放弃更合理的手段。

### P-B09 刚立原则就亲手打脸

```text
他前一章还说绝不碰灰色交易。
这一章为了快一点，就很自然地答应了。
```

问题：价值底线翻得太轻。

## D 组：反转和误会硬翻

### P-B10 最后一句强翻

```text
所有人都误会了她。
最后一句才说其实监控坏了，是另一个人做的。
```

问题：前面没种，后面硬翻。

### P-B11 角色集体失明型误会

```text
明明一句话就能解释清的事，所有人偏偏三十章都不问。
```

问题：误会靠集体降智维持。

### P-B12 反派忽然自曝全部

```text
他眼看就要赢了，却忽然把所有计划完整说了一遍。
```

问题：反转靠反派配合，不靠前文埋伏。

## E 组：发生了但没变化

### P-B13 吵完像没吵

```text
他们大吵一架。
第二天像什么都没发生过一样继续相处。
```

问题：没有后果，剧情白跑。

### P-B14 掉马后无人改口

```text
她真实身份曝光了。
可所有人的态度、资源和风险都没有任何变化。
```

问题：大事件没有改写局面。

### P-B15 赢了也没代价

```text
她当众打脸成功，既没有树敌，也没有后续麻烦，还顺便解决了一切。
```

问题：只有爽点，没有后续成本，世界像纸板做的。
