# 去 AI 味正例库

使用方法：

1. 先判断自己卡在“动作化 / 场内变化 / 人物声音 / 节奏”哪一层。
2. 组内挑 `2-4` 个最接近的例子。
3. 借的是承载方式，不是句面。

## 快速索引

- A 组：用动作和物件承接情绪
- B 组：用局面和细节立住压迫感
- C 组：用身份和立场分开人物声音
- D 组：用节奏和短句打掉解释腔

## A 组：用动作和物件承接情绪

### AAV-G01 放下物件代替“五味杂陈”

```text
她张了张嘴，最后只把手里的杯子放回桌上。陶瓷磕出一声脆响。
```

该学什么：不直接解释复杂情绪，而是让迟疑和手上动作把情绪托住。

### AAV-G02 眼神绕开，比“难过”更准

```text
她没再看他，低头去抠纸袋边角，抠得那层白纸起了毛。
```

该学什么：回避动作和无意义小动作，能把情绪落到身体上。

### AAV-G03 想说又忍回去

```text
他“嗯”了一声，像是还想补一句，最后只把手机扣在桌面上。
```

该学什么：情绪不一定靠爆发，也可以靠被按住的后半句。

### AAV-G04 借环境小声放大尴尬

```text
谁都没说话，只有空调叶片轻轻打着颤，嗒嗒地响。
```

该学什么：不是写“气氛凝固”，而是写安静之后什么声音突然被听见。

### AAV-G05 一个退让动作写出关系落差

```text
她往后让了半步，后腰正好抵上料理台边。
```

该学什么：关系里的弱势和紧张，先看身体怎么退。

## B 组：用局面和细节立住压迫感

### AAV-G06 不喊也能压住场

```text
他没抬高声音，只是把打火机“啪”地扣上，屋里就安静了。
```

该学什么：危险感最好通过别人被迫收声来体现。

### AAV-G07 一个小动作改写主次

```text
她说完那句话后，他没有接，只伸手把她面前的水杯推远了半寸。
```

该学什么：关系变化可以藏在距离调整里，不必总靠宣告。

### AAV-G08 先做，再让别人反应

```text
他把合同翻到最后一页，指尖压住签名处，才抬眼看她。
```

该学什么：压迫感经常来自“动作已先行”，不是来自大词。

### AAV-G09 场内秩序被重新排位

```text
她一落座，原本正在说话的两个人都停了，连服务员都先看了她一眼。
```

该学什么：人物分量可以靠别人反应间接立住。

### AAV-G10 危险感来自后果预感

```text
他把袖口慢条斯理地折上去时，旁边的人已经往后撤了椅子。
```

该学什么：不要只写“危险”，要写别人已经开始准备挨后果。

## C 组：用身份和立场分开人物声音

### AAV-G11 长辈借规矩压人

```text
「昭儿，我平日是如何教你的？和离岂有将嫁妆尽数带回的道理？」
```

该学什么：长辈的压法是借辈分和规矩，不是泛泛说“你不对”。

### AAV-G12 身份与退路一起砍

```text
「别的千金小姐离婚了还有娘家，你一个假千金离婚了还有什么？」
```

该学什么：一句话同时压身份、资源、退路，声音立刻就分开了。

### AAV-G13 孩子口吻比大人更扎

```text
「妈妈，你为什么要和爸爸离婚？」
```

该学什么：人物声音的差异，不只靠措辞，也靠说话者的位置。

### AAV-G14 冷静宣布行动

```text
「下周三，我们去领离婚证。」
```

该学什么：真有压的台词往往带行动安排，不靠铺陈情绪。

### AAV-G15 极短台词带身份感

```text
「账给我。」
```

该学什么：有些角色越不解释，身份越清楚。

## D 组：用节奏和短句打掉解释腔

### AAV-G16 长句之后插一刀

```text
她原本还想把话说圆，看到他抬眼的那一瞬，后半句直接断了。
没有了。
```

该学什么：短句能把前面的铺垫一下切出刀口。

### AAV-G17 让一句话说不完

```text
「我不是那个意思，我只是觉得你今天这样有点」
她自己先停了。
```

该学什么：被自己噎住，比完整解释更像人。

### AAV-G18 动作截断说明腔

```text
他开口前先拽松了领口，像是把那口气硬压了回去。
```

该学什么：先给一个动作缓冲，能把后面的话从“说明”变成“难以出口”。

### AAV-G19 让停顿参与句子

```text
「行。」他说。
过了两秒，又补了一句：「你别后悔。」
```

该学什么：停顿是节奏的一部分，不必所有内容一次说满。

### AAV-G20 留一处空白给读者补

```text
她看着那串早就删掉又重新加回来的备注，没吭声。
```

该学什么：有些信息不用解释，留给读者自动连线，反而更真。
