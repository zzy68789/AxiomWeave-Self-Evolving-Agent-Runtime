# 去 AI 味反例库

## 快速索引

- A 组：空泛总结
- B 组：形容词和氛围套话
- C 组：说明书对白
- D 组：平均工整腔

## A 组：空泛总结

### AAV-B01

```text
她的内心翻涌着复杂情绪。
```

问题：只有作者判断，没有现场承载。

### AAV-B02

```text
她心中五味杂陈，一时间不知道该说什么。
```

问题：复杂情绪被一句总结盖掉了，读者看不到它怎么来的。

### AAV-B03

```text
他此刻的心情十分沉重。
```

问题：只报情绪结果，不给触发点和身体反馈。

### AAV-B04

```text
她感慨万千，思绪纷飞。
```

问题：词很多，内容没有落点。

### AAV-B05

```text
他们两人都陷入了复杂而微妙的情绪之中。
```

问题：把两个人一起概括掉了，谁的反应都没保住。

## B 组：形容词和氛围套话

### AAV-B06

```text
空气仿佛凝固了。
```

问题：气氛被直接判定，没有写出是谁让场子冷下来。

### AAV-B07

```text
时间仿佛在这一刻静止。
```

问题：大而空，常见但很少真的有画面。

### AAV-B08

```text
他是一个清冷、危险、深邃、低沉又充满磁性的男人。
```

问题：标签堆叠，像角色说明卡，不像正文现场。

### AAV-B09

```text
两人之间的气氛非常微妙。
```

问题：作者替读者下结论，关系没有通过动作体现。

### AAV-B10

```text
她身上散发着一种说不清道不明的破碎感。
```

问题：抽象感受再次套抽象词，依然没有具体物。

## C 组：说明书对白

### AAV-B11

```text
「你忘了吗？三年前你父亲因为家族企业危机，所以才让我嫁给你，这一切都是因为……」
```

问题：人物在冲突场里替作者补背景。

### AAV-B12

```text
「我现在非常伤心，也非常失望，因为你今天的行为深深地伤害了我。」
```

问题：情绪被完整解释，像心理课复盘。

### AAV-B13

```text
「我之所以这么做，是因为我从小缺乏安全感，所以我才会这样试探你。」
```

问题：人物像在提交自我分析报告。

### AAV-B14

```text
「请你尊重我的感受，并认真思考我们的关系。」
```

问题：谁都能说这句话，没有身份感，也没有刀口。

### AAV-B15

```text
「其实我们之间的问题主要来源于沟通不畅和价值观差异。」
```

问题：像会议纪要，不像人吵架。

## D 组：平均工整腔

### AAV-B16

```text
他愤怒地看着她，然后用一种低沉而克制的声音说出了一大段完整而清晰的话。
```

问题：高压时刻过于完整，像为朗读准备的句子。

### AAV-B17

```text
她先是震惊，随后委屈，紧接着难过，最后陷入沉默。
```

问题：像流程图列情绪，没有一个环节真正被看见。

### AAV-B18

```text
她的眼中闪过一丝复杂的情绪，但很快恢复了平静。
```

问题：这是万能占位句，几乎可以套到任何场合。

### AAV-B19

```text
他们彼此对视，谁都没有说话，仿佛一切尽在不言中。
```

问题：又回到作者总结，具体信息仍然缺席。

### AAV-B20

```text
她轻轻地点了点头，优雅地转身离开，只留下一个意味深长的背影。
```

问题：动作存在，但全是模板化包装词，还是发虚。

## 补充提醒

遇到这类句子，不要只换同义词。

更有效的做法通常是：

- 删掉总结
- 换成动作或物件
- 把解释挪给局面变化
- 让人物用自己的身份说话
