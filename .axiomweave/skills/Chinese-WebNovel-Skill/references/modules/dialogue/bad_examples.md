# 对白反例库

## 快速索引

- A 组：空话套话
- B 组：情绪大但没有刀口
- C 组：说明书对白
- D 组：人物声音全部一样

## A 组：空话套话

### D-B01 占位词连发

```text
「你怎么能这样？」
「我不敢相信。」
「你太让我失望了。」
```

问题：每句都像占位词，没有具体伤害点。

### D-B02 只会说“你变了”

```text
「你变了，真的变了。」
```

问题：情绪有了，但没有任何可落地的矛盾内容。

### D-B03 万能难过句

```text
「我现在真的不知道该说什么。」
```

问题：看似克制，实则什么也没推进。

### D-B04 全是空洞指责

```text
「你太过分了！你根本不是人！」
```

问题：声量很大，但没有具体刀口。

## B 组：情绪大但没有刀口

### D-B05 只有情绪宣告

```text
「我真的很难过！你根本不懂我！我快疯了！」
```

问题：全是情绪宣告，没有关系和利益变化。

### D-B06 只会喊恨

```text
「我恨你！我讨厌你！我再也不想见到你！」
```

问题：不断重复强情绪词，却没有一句真正伤到对方。

### D-B07 大哭式控诉

```text
「你知不知道我这些年有多痛苦！有多委屈！有多崩溃！」
```

问题：这类句子只是在抛情绪总量，不是在制造局面变化。

### D-B08 互吼但谁也没赢

```text
「你闭嘴！」
「你才闭嘴！」
```

问题：表面吵起来了，实则没有任何新信息和权力变化。

## C 组：说明书对白

### D-B09 背景讲义型

```text
「你忘了吗？三年前你父亲因为家族企业危机，所以才让我嫁给你，这一切都是因为……」
```

问题：人物在吵架，不是在给读者补课。

### D-B10 情绪分析报告型

```text
「我现在非常伤心，也非常失望，因为你今天的行为深深地伤害了我。」
```

问题：像心理课复盘，不像冲突现场。

### D-B11 自我说明书型

```text
「我之所以这么做，是因为我从小缺乏安全感，所以我才会这样试探你。」
```

问题：人物像在提交自我分析报告。

### D-B12 会议纪要型

```text
「其实我们之间的问题主要来源于沟通不畅和价值观差异。」
```

问题：像会议总结，不像人吵架。

## D 组：人物声音全部一样

### D-B13 同一句谁说都一样

```text
长辈说：「请你尊重我的感受。」
校霸说：「请你尊重我的感受。」
保姆说：「请你尊重我的感受。」
```

问题：谁说都一个口气，人不见了。

### D-B14 高压场景过于工整

```text
他怒不可遏地说了一大段完整而工整的话。
```

问题：高压场景里，过于工整往往是假。

### D-B15 人人都像懂心理学

```text
「你现在是在逃避亲密关系。」
「不，你是在进行情绪勒索。」
```

问题：像两个分析师，不像具体人物。

### D-B16 身份差完全消失

```text
大小姐、司机、初中生都用同一种礼貌克制的长句表达不满。
```

问题：身份、教养、年龄、处境都没进入台词。
