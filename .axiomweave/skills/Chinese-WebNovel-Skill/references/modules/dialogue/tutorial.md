# 网文对白教程

## 一、先统一一个定义

好的对白不是聊天记录，也不是情绪宣泄。

它至少要同时带出三样东西：

- 关系位置
- 现实利益
- 情绪压迫

如果一句话里什么都没有，只剩“我很生气”“你误会我了”，那通常就是空。

## 二、对白真正要完成什么

### 1. 施压

对白要让场上的压力变得可听见。

### 2. 放信息

信息最好嵌在对抗里出来，而不是靠人物轮流做讲解员。

### 3. 改关系

一句话说完，双方位置最好发生一点偏移。

### 4. 立人

同样一句拒绝，不同身份、不同教养、不同脾气的人，说法不会一样。

## 三、最常见的四类高张力对白

### 1. 地位压迫型

说话人借长辈、上位者、制度、家规、身份施压。

### 2. 情感绑架型

说话人用亲密关系、亏欠感、孩子、旧情、眼泪来施压。

### 3. 利益摊牌型

说话人把钱、名声、婚姻、资源、退路直接摆上桌。

### 4. 冷静判决型

不大吵，但一句话直接改局面。

## 四、判断对白有没有力，看这五件事

### 1. 说这句话的人现在想达到什么

没有目的的对白，通常就是闲聊。

### 2. 这句话里有没有新东西

可以是新信息、新站位、新态度、新威胁。

### 3. 这句话说完，关系有没有偏移

哪怕只偏一点，也比原地复读强。

### 4. 这句话是不是只有情绪，没有刀口

“你怎么能这样”太空，因为没有具体刀口。

### 5. 换个人说，这句台词还能成立吗

如果谁说都一样，说明人物声音没立住。

## 五、对白最常见的五种失败

### 1. 空话套话

比如：

- “你怎么能这样”
- “我不敢相信”
- “你误会我了”

### 2. 角色边吵边讲设定

像两个人轮流念大纲。

### 3. 情绪很大，但没有利益和关系变化

只会喊，不会压。

### 4. 说话口径全部一样

长辈、富二代、学生、保姆、上位者，全都一个语气。

### 5. 对白过长，把张力说散了

高压场面里，短句常常更有刀。

## 六、写对白前的最小计划

动笔前先写清这四句：

- 这一轮对话谁想赢什么：
- 他最想戳对方哪一刀：
- 对方最怕被戳什么：
- 这轮对话结束后，关系或局面要怎么偏：

如果这四句说不清，对白多半会空转。
