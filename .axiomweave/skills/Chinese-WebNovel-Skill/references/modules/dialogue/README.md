# 对白模块

这是一个专门处理网文对白问题的专项模块。

它主要解决：

- 对白很空
- 对白像说明书
- 角色说话都一个腔调
- 有情绪，没有信息
- 有信息，没有压力
- 对白不能推动关系和局面

## 什么时候调用

当任务明确落在这些问题上时，优先调用本模块：

- 用户要写对白、改对白、补高张力对白
- 用户说“台词假”“都像 AI 在说话”
- 关系戏不够刺
- 冲突戏不够压

## 建议阅读顺序

1. 先读 [tutorial.md](tutorial.md)，统一对白的目标。
2. 再读 [runtime.md](runtime.md)，判断问题卡在哪一层。
3. 去 [good_examples.md](good_examples.md) 选 `2-4` 个正例。
4. 去 [bad_examples.md](bad_examples.md) 选 `1-2` 个反例。
5. 先写对白压力计划，再落正文。

## 文件说明

- [tutorial.md](tutorial.md)
  把对白从“说得像不像人”提升为“关系、利益、情绪如何同时上桌”。
- [runtime.md](runtime.md)
  给模型的执行顺序。
- [good_examples.md](good_examples.md)
  以现有语料中的高张力对白摘录为主。
- [bad_examples.md](bad_examples.md)
  汇总最常见的空台词和假台词错误。
- [source_index.md](source_index.md)
  标明本模块引用的主 skill 段落和本地语料来源。
